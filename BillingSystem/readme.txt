Prerequisites to run this Project and setup Development environment:
1.Java JDK
2.STS
3.MySql
TOOLS Download URL : https://docs.google.com/spreadsheets/d/1n4F6EiAEbw-ro4g5zjkErBtHOiOiFf3hCRq3t0fOncw/edit?usp=sharing

Steps to start this Project:
Step-1 : Take a Checkout/Download this project.
Step-2 : Start STS and import as existing maven project.
Step-3 : Create a DB in MySql.
Step-4 : Import spsecbasic.sql to MySql
Step-5 : Make db changes in application.properties file.
Step-6 : Run BillingSystemApplication.java file to start the project.
